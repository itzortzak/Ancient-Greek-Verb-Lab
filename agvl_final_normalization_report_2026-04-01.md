## Main philological decisions

### στρώννυμι

The lemma is now closed as **fully audited**. The visible manual paradigm no longer carries parallel forms in the same slot. The display policy is:

- primary future active: **στρώσω**
- primary active aorist: **ἐστόρεσα**
- primary perfect active: **ἔστρωκα**
- primary perfect middle/passive: **ἔστρωμαι**

The multistem tradition is still preserved, but no longer as duplicate display rows. The following source-backed alternatives are now retained as **recognition-only** forms in the form index and in metadata:

- στρωννύω, στορέννυμι, στόρνυμι
- στορέσω, στορῶ
- ἔστρωσα
- στρῶσαι
- στρώσας, στρώσασα, στρῶσαν
- ἐστρωσάμην, ἐστορεσάμην
- στρωθήσομαι, ἐστρώθην, ἐστορέσθην
- ἐστόρεσμαι, ἐστορέσμην

### ῥήγνυμι

The only remaining internal slot conflict in the recently audited set was normalized. The visible paradigm now keeps **ἔρρηχα** as the perfect active 1sg, while **ἔρρωγα** remains available as a recognition-only source-backed variant.

These are not automatically errors. In almost all cases they are older rich paradigms that intentionally preserve alternative historical forms. The scan therefore distinguishes between:

- **blocking conflicts**: duplicate rows in the same slot that should be normalized for classroom display
- **non-blocking alternations**: documented parallel forms kept intentionally

In the present pass, only two blocking issues were resolved: **στρώννυμι** and **ῥήγνυμι**.
