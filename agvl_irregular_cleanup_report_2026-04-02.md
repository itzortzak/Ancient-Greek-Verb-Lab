# AGVL irregular cleanup patch — 2026-04-02

## Τι έγινε

Έγινε νέο runtime pass ελέγχου, διόρθωσης και συμπλήρωσης για **7 υψηλής προτεραιότητας ανώμαλα λήμματα**:

- εἰμί
- γίγνομαι
- λέγω
- δίδωμι
- τίθημι
- ἵστημι
- ἵημι

Κεντρικός στόχος αυτού του pass ήταν η **μείωση των διπλών / αντικρουόμενων εμφανίσεων στην ενότητα «Κλίση»**, χωρίς να χαθούν οι εναλλακτικοί τεκμηριωμένοι τύποι από την αναγνώριση.

## Αποτέλεσμα

- στοχευμένα λήμματα: **7**
- visible entries μετά το cleanup: **2800**
- demoted recognition-only entries: **726**

## Βασική αρχή του patch

Η ορατή κλίση δείχνει πλέον τον **σχολικό ή φιλολογικά ασφαλέστερο πυρήνα** ανά slot.
Οι υπόλοιπες τεκμηριωμένες παραλλαγές **δεν διαγράφτηκαν**· πέρασαν σε recognition-only επίπεδο στο `FORM_INDEX`, ώστε να συνεχίσουν να αναγνωρίζονται χωρίς να επιβαρύνουν την ορατή κλίση.

## Κατά λήμμα

### εἰμί
- σχολικοποιήθηκε η ορατή κλίση
- αφαιρέθηκαν από την ορατή προβολή επικοί / ιωνικοί παράλληλοι τύποι
- κρατήθηκαν visible τύποι όπως: `ἐστί(ν)`, `εἰσί(ν)`, `εἴημεν / εἶμεν`, `ἔσῃ / ἔσει`, `ἔστων / ὄντων / ἔστωσαν`

### γίγνομαι
- προτιμήθηκαν οι αυξημένοι αοριστικοί τύποι στην ορατή κλίση
- κανονικοποιήθηκαν optional-ν και παράλληλες ορθογραφίες
- διατηρήθηκαν visible τύποι όπως: `ἐγενόμην`, `ἐγενήθην`, `γέγονε(ν)`, `γεγόνασι(ν)`

### λέγω
- η ορατή κλίση μεταφέρθηκε στον σχολικό suppletive άξονα
- visible: `ἐρῶ`, `εἶπον`, `ἐρρήθην`, `εἴρημαι`, `εἰπεῖν`, `εἰπών`
- demoted στην αναγνώριση: `λέξω`-series, `ἔλεξα`-series, `λεχθήσομαι`-series, `ἐλέχθην`-series, `λέξαι`, `λέξας`, `λέλεγμαι`

### δίδωμι
- καθαρίστηκαν από την ορατή κλίση οι διδώσω-series, οι μη αυξημένοι και οι αρχαιότεροι αναλογικοί τύποι
- visible: `δώσω`, `δώσομαι`, `ἔδωκα`, `δῶ`, `δοῦναι`, `ἐδόμην`, `ἐδόθην`, `δέδωκε(ν)`

### τίθημι
- κανονικοποιήθηκε η visible προβολή στον άξονα `τέθηκα / τέθημαι`
- οι `τέθεικα / τέθειμαι` σειρές μεταφέρθηκαν σε recognition-only επίπεδο
- τα plural optatives συμπτύχθηκαν σε ενιαίες visible εγγραφές, π.χ. `τιθείημεν / τιθεῖμεν`

### ἵστημι
- **δεν αφαιρέθηκαν** οι δύο φιλολογικά τεκμηριωμένοι άξονες
- ο transitive/sigmatic και ο intransitive/stative άξονας διατηρούνται visible, αλλά πλέον συμπιεσμένοι σε ένα slot, π.χ. `ἔστησα / ἔστην`, `στήσω / στῶ`, `ἑστηκώς / ἑστώς`

### ἵημι
- συμπτύχθηκαν οι κανονικές διπλές σχολικές εμφανίσεις, π.χ. `ἵης / ἱεῖς`, `ἵην / ἵειν`, `εἵημεν / εἷμεν`

## Τεχνικές αλλαγές

- προστέθηκε νέο runtime patch: `data/agvl_runtime_patch_irregular_cleanup.js`
- ενημερώθηκε το `index.html` ώστε να φορτώνει το patch μετά το `agvl_runtime_patch_participles.js`
- καθαρίστηκε το `FORM_INDEX` για τα στοχευμένα λήμματα και ξαναχτίστηκε με visible + recognition-only εγγραφές

## Επιφύλαξη

Το patch **δεν** επιχειρεί να εξαφανίσει όλες τις φιλολογικά πραγματικές παραλληλίες. Όπου υπάρχουν δύο σοβαρά τεκμηριωμένοι μορφολογικοί άξονες, ιδίως στο **ἵστημι**, αυτοί διατηρούνται, αλλά χωρίς την προηγούμενη χαοτική πολλαπλή προβολή.
