# Release Notes — v1.0.0 Release Candidate

This release candidate consolidates the work completed in earlier development stages and prepares the project for stable GitHub publication.

## Highlights
- school-oriented interface for verb analysis and conjugation
- contracted verbs displayed in contracted form
- multiple-analysis handling for pedagogically important ambiguous forms
- manual paradigms for core difficult lemmas
- spot-check and slot-by-slot validation layers
- classroom test pass and targeted failure review
- source transparency with differentiated integration statuses

## Packaging changes
- backup files removed from the distributed package
- documentation normalized for repository publication
- release identity clarified as `v1.0.0 Release Candidate`

## Recommendation
This package is suitable for public repository publication and controlled classroom use, with the usual philological caution for difficult or highly ambiguous forms.
