# SLOT-BY-SLOT VALIDATION

Σε αυτή τη φάση επεκτάθηκε το χειροκίνητα επιμελημένο layer slot-by-slot validation για τα λήμματα:
- ἀποκρίνομαι
- πορεύομαι
- φημί
- πάσχω
- πίπτω
- ἀποθνῄσκω
- βούλομαι
- δύναμαι
- μανθάνω
- λανθάνω
- ὁράω
- δίδωμι
- τίθημι
- ἵστημι
- λαμβάνω
- γίγνομαι
- λέγω
- ἔχω
- φέρω
- μάχομαι

Το layer αυτό δεν αντικαθιστά την πλήρη κλιτική βάση. Επισημαίνει συγκεκριμένα σχολικά κρίσιμα slots που έχουν ελεγχθεί χειροκίνητα και προβάλλονται ρητά στην ενότητα «Κλίση».
