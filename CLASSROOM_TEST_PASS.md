# CLASSROOM_TEST_PASS

Η έκδοση αυτή προσθέτει στοχευμένο `classroom test pass` με αντιπροσωπευτικούς σχολικούς ρηματικούς τύπους.

Σκοπός:
- να ελεγχθεί η εφαρμογή σε τύπους που εμφανίζονται συχνά στη σχολική πράξη,
- να φανεί αν το active morphology layer αποδίδει τη βασική αναγνώριση,
- να καταγραφούν λήμματα που χρειάζονται επόμενο φιλολογικό έλεγχο.

Το test set καλύπτει:
- κανονικά ρήματα,
- συνηρημένα,
- αποθετικά,
- β΄ αορίστους,
- ρήματα σε -μι,
- ανώμαλα υψηλής συχνότητας.
