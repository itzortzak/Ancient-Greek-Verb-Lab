# Coverage statement

This release contains a mixed morphology environment intended for school use.

## Coverage layers

The application currently combines:
- a school-core verb layer
- imported morphology data
- manually curated paradigms for difficult high-frequency lemmas
- targeted philology hardening
- spot-check and slot-validation layers for selected difficult verbs
- a multiple-analysis layer for school-relevant ambiguous forms

## Interpretation of coverage

Coverage in the library does not automatically imply the same level of philological control for every lemma.

For classroom use, the safest path is:
1. canonical school-core lemmas
2. manually curated paradigms
3. spot-checked and slot-validated difficult lemmas
4. broader imported morphology as extended support

## Current release stance

This package should be treated as a strong school-oriented release candidate rather than a fully exhaustive historical morphology system.
